# 2026-06-26 코틀린 기초 문법 학습 기록

## 📚 오늘의 학습 내용 요약

### 1. 변수 선언 (var, val) 및 타입 시스템
- `var`는 변수, `val`은 상수로 선언하며, 코틀린은 엄격한 타입 지정 필요
- `Double`, `Float`에 `Int` 값을 직접 할당 불가능하며, 명시적 타입 지정 필요 (예: `20.0F`, `13L`)

### 2. lateinit과 Any 타입
- `lateinit var`는 기초 데이터 타입(Int, Char, Float, Double, Boolean, Byte, Short, Long)에 사용 불가
- `Any` 타입은 모든 타입의 값을 할당 가능하며, `.toInt()` 등의 메서드로 명시적 타입 변환 가능

### 3. 함수와 기본 매개변수
- 함수는 `fun 함수명(매개변수): 반환타입 { }` 형식으로 선언
- 반환값이 없는 경우 `Unit` 타입을 사용하며 생략 가능
- 함수 매개변수에 기본값을 지정하면 가변 인수 전달 가능

### 4. 제어문 (if, when)
- `if`는 일반 프로그래밍 언어와 동일하나, 삼항 연산자처럼 값 할당 가능
- `when` 제어문은 switch보다 강력하여 범위(`in 1..10`), 타입 검사(`is String`), 다중 값 비교 가능

### 5. 반복문과 실제 응용
- `for (i in 1..n)` 형식으로 범위 반복 가능
- 반복문을 활용한 수열 계산으로 원주율(π)을 근사적으로 계산하는 방식 학습 (라이프니츠 급수)
